Crimson Tomato
==============

## Introduction

The time management application related to [Pomodoro Technique](https://en.wikipedia.org/wiki/Pomodoro_Technique).

Features:

* Sync periods to local calendar.
* Export records to CSV file.
* Task management.
* Vibrate when finished.
